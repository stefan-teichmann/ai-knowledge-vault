### Praktische Übung

### Hello world! Edge Function

1. Geben Sie in Lovable folgenden Prompt ein: „Erstelle eine Supabase Edge Function, die 'Hello world' zurückgibt. Die Edge Function soll durch einen roten Button im Dashboard ausgelöst werden und das Ergebnis direkt neben dem Button anzeigen.“
    
2. Dies sollte einfach umzusetzen sein.
    
3. Erweitern Sie die Übung mit folgendem Prompt: „Anstelle des roten Buttons möchte ich zwei Felder im Dashboard sehen, in die ich eine Zahl eingeben kann. Beide Zahlen werden von der Edge Function addiert und das Ergebnis wird angezeigt.“

##  Edge Funktion Deployment 04:34

### Praktische Übung

### Unser erstes richtiges Feature

1. Geben Sie in Lovable folgenden Prompt ein: „Lass uns das erste Feature bauen: Der Benutzer soll einzelne Felder eingeben können (Länge, Breite, aktuelle Bepflanzung, Dauer der aktuellen Bepflanzung/Erntezeit). Die Feldeingabe soll in einem Modal erfolgen, das sich öffnet, wenn der Benutzer im Dashboard auf ‚Erstes Feld erstellen‘ klickt. Die eingegebenen Felder sollen für den Benutzer in einer neuen Tabelle in der Datenbank mithilfe einer Edge Function gespeichert werden. Es soll drei Unterseiten im Dashboard geben: Dashboard-Startseite, Feldansicht und den Nachsaat-Kalender. Erweitere die Dashboard-Startseite, um die oben genannten Funktionen zu implementieren, und füge eine Feldansicht hinzu, in der die aus der Datenbank erstellten Felder mithilfe einer Edge Function angezeigt werden.“
    
2. Iterieren Sie nun einfach so lange durch den Flow, bis es Ihnen passt oder sinnvoll erscheint.
    
3. Am Ende des Prompts zeigt Ihnen Lovable einen Link zu den Logdateien der Edge Functions an. Hier können Sie klicken und sehen, welche Fehler beim Aufrufen der Funktionen aufgetreten sind (z. B. „Create Field Function Logs“ oder „Get Fields Function Logs“).
