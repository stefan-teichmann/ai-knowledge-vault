### Was ist Lovable?

- Lovable ist ein No-Code-Tool, das Ideen in Minuten in Live-Websites verwandelt.    
- Es ermöglicht die Erstellung von Backends, Datenbanken und Publikationen über natürliche Sprache.    
- Das Tool wird als „persönlicher Full-Stack-Engineer“ beschrieben.
    

### Warum Lovable nutzen?

- Traditionelle Implementierungen erfordern oft technisches Wissen oder hohe Investitionen.    
- Lovable macht die App-Entwicklung für Agenturen, Unternehmen, Kreative und Entwickler zugänglich.
    
- **Workflow-Vergleich:**    
    - **Früher:** Lange Phasen für Design, Budgetierung und endlose Feedbackschleifen.        
    - **Mit Lovable:** Erstellung eines Proof of Concept (PoC) in ca. 2 Stunden und Feedbackschleifen zum Preis von wenigen Prompts.
        

### Erste Schritte mit Lovable

1. **Account erstellen:** Über lovable.dev mit GitHub, Google oder E-Mail registrieren.  
2. **Kosten:** Der Free Plan bietet ca. 5 Prompts pro Tag. Für den Kurs wird ein „Starter“-Account (25 $/Monat) empfohlen.    
3. **Erstes Projekt:** Erstellung einer einfachen „Hello World“-App über einen Prompt.
    

### Fortgeschrittene Funktionen: Supabase & Auth

- **Supabase:** Lovable nutzt Supabase als Standard-Datenbank (PostgreSQL) und für die Authentifizierung.    
- **Authentifizierung:** Ermöglicht Nutzern das Registrieren und Einloggen, um auf persönliche Daten zuzugreifen.    
- **Row Level Security (RLS):** Ein Sicherheitskonzept, das definiert, wer auf welche Zeilen einer Datenbanktabelle zugreifen darf (z. B. Nutzer dürfen nur ihre eigenen Profile sehen).
    

### Edge Functions

- Edge Functions agieren als Server für dynamischen Code.    
- Sie sind wichtig für Sicherheitsaspekte, wie das Verbergen von API-Keys (z. B. für OpenAI-Anbindungen).    
- Lovable übernimmt das Deployment dieser Funktionen automatisch auf Supabase.
    

### Go-Live und Versionierung

- **GitHub-Integration:** Projekte können mit einem GitHub-Repository verbunden werden, um volle Kontrolle über den Code zu behalten.    
- **Domains:** Eigene Domains können direkt über Lovable erworben oder bestehende (z. B. von GoDaddy) verbunden werden.    
- **Publishing:** Mit einem Klick auf „Publish“ wird die Web-App live geschaltet.    
- **Analytics:** Lovable bietet eigene Analytics-Tools zur Echtzeit-Überwachung der App-Nutzung.
