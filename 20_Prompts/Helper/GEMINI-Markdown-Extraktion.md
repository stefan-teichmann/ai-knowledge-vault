# Prompt: GEMINI-Markdown-Extraktion

Extrahiere mir aus den im folgenden hochgeladenen PDFs oder Bildern eine Markdown-Formatierte Texte.
Überschriften sollten auf der dritten Ebene beginnen (`### `).

Es soll jeweils pro Frage nur die zuletzt hochgeladenen Dateien verwendet werden.
Es sollen keine weiteren Rückfragen gestellt werden.
An die Anforderungen soll sich gehalten werden.
